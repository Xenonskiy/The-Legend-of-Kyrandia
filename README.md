# The Legend of Kyrandia Remake

<h3>This is a remake of the first part of the game The Legend of Kyrandia. In the game, you will be able to switch between original and enhanced textures, as well as original and improved music. You can change the language and voiceovers. The inventory has been expanded. You can enable the game map and see which items you left in which location. Additionally, you can enable animations for the locations.</h3>


<h3>Это ремейк первой части игры The Legend of Kyrandia. В игре можно будет менять оригинальные и улучшенные текстуры. Оригинальную и улучшенную музыку. Менять язык и озвучку. Увеличен инвентарь. Можно будет включать карту игры и видеть какие предметы в какой локации оставил. Также можно будет включать анимацию для локаций.</h3>
